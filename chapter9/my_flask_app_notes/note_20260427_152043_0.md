# my_flask_app 项目代码结构探索

## 当前目录状态
`my_flask_app/` 目录目前是**空目录**，没有任何文件。它是作为项目演示框架的一部分存在的。

## 父级项目的整体结构

项目根目录（`../`）包含以下内容：

### 核心代码库（codebase/）
一个完整的 Python 模块包，包含：
- `__init__.py` - 包初始化，版本号 1.0.0
- `data_processor.py` - 数据处理模块（pandas 数据清洗、转换、聚合、导出）
- `models.py` - 数据模型模块（User, Product, Order, Transaction 四个 dataclass）
- `utils.py` - 工具函数模块（配置加载、时间戳、目录创建、邮箱验证等）
- `api_client.py` - API 客户端模块（支持 GET/POST/PUT/DELETE 请求）

### 主程序入口
- `main.py` - 简单的应用入口，打印启动信息
- `codebase_maintainer.py` - 完整的 CodebaseMaintainer 智能体实现（整合 ContextBuilder + NoteTool + TerminalTool + MemoryTool）

### 示例脚本
6个教学示例脚本，逐步演示工具使用：
1. `01_context_builder_basic.py` - 上下文构建基础
2. `02_context_builder_with_agent.py` - 上下文+智能体
3. `03_note_tool_operations.py` - 笔记工具操作
4. `04_note_tool_integration.py` - 笔记工具集成
5. `05_terminal_tool_examples.py` - 终端工具示例
6. `06_three_day_workflow.py` - 三天工作流示例

### 其他目录
- `project/` - 另一个项目演示目录（也有 main.py 和 README.md）
- `data/` - 数据文件（sales_2024.csv）
- `data_pipeline_refactoring_notes/` - 数据管道重构笔记
- `kb/`、`knowledge_base/` - 知识库目录（当前为空）
- `logs/` - 日志目录（当前为空）
- `memory_data/` - 记忆数据（memory.db）
- `my_flask_app_notes/` - 本项目的笔记索引（notes_index.json）
- `project_notes/` - 项目笔记

### 配置文件
- `.env` - 环境变量配置
- `README.md` - 项目说明

## 技术栈
- Python 3
- pandas（数据处理）
- requests（HTTP 客户端）
- hello_agents（智能体框架）
- dataclass（数据模型）
