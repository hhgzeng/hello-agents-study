### 🟡 Day 2 — API 客户端重构（2026-04-29 周二）

#### Task 2.1：提取统一请求方法 ⏱ 90min

- [ ] 实现 `_request(method, endpoint, **kwargs)` 统一入口
- [ ] 为 session 添加默认 `timeout=30`
- [ ] 添加指数退避重试（3 次：1s → 2s → 4s）
- [ ] 添加自定义 `User-Agent` 头

#### Task 2.2：全面异常处理 ⏱ 60min
- [ ] 捕获 `ConnectionError` → 转为 `APINetworkError`
- [ ] 捕获 `Timeout` → 转为 `APITimeoutError`
- [ ] 捕获 4xx → 转为 `APIHTTPError`（含响应体前 200 字符）
- [ ] 捕获 5xx → 重试 + 最终转为 `APIHTTPError`
- [ ] 捕获 `JSONDecodeError` → 转为 `APIClientError`

#### Task 2.3：DELETE 安全机制 ⏱ 30min
- [ ] 添加 `confirm: bool = False` 参数

#### Task 2.4：编写 API 客户端测试 ⏱ 60min
