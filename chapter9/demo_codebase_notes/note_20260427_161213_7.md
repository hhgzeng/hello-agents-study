# demo_codebase 项目结构分析

## 项目概览

**项目名称**: demo_codebase（Python包）
**版本**: 1.0.0
**结构类型**: 扁平单层结构（无子目录）
**包文件数**: 5个模块文件

## 模块职责

| 文件 | 职责 | 核心组件 |
|------|------|---------|
| `__init__.py` | 包初始化 | 声明版本号 `1.0.0` |
| `api_client.py` | API通信 | `APIClient` 类（GET/POST/PUT/DELETE） |
| `data_processor.py` | 数据处理 | 5个函数：process/clean/transform/aggregate/export |
| `models.py` | 数据模型 | 4个dataclass：User, Product, Order, Transaction |
| `utils.py` | 工具函数 | 6个工具函数：config加载/保存、时间戳、目录创建、大小格式化、邮箱验证 |

## 技术栈

- Python 3（使用类型提示、dataclass）
- `requests`（HTTP客户端）
- `pandas`（数据处理）
- `re`（正则验证）
